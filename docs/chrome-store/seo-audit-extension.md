# Chrome Store Draft: SEO Page Audit Analyzer

## Short Description
Audit current page SEO basics and prepare AI repair suggestions.

## Long Description
AutoPub SEO Page Audit Analyzer checks title, description, headings, image alt text, canonical URL, and other page SEO basics. Pro features can add AI repair suggestions and report export workflows.

## Permissions Rationale
- `activeTab`: audit the current page after user action.
- `scripting`: support current-page inspection.
- `storage`: save session and settings.
- Host permissions: inspect page metadata and visible document structure.
