# Chrome Store Draft: Visual Saver & AI Analyzer

## Short Description
Scan page images, save visual assets, and run credits-gated AI visual analysis.

## Long Description
AutoPub Visual Saver & AI Analyzer helps creators, ecommerce operators, and design teams collect visible page images, review image metadata, download assets, and prepare AI prompt or style analysis workflows.

## Permissions Rationale
- `activeTab`: scan the current page after user action.
- `scripting`: support current-page inspection.
- `storage`: save session and settings.
- `downloads`: download selected visible image assets.
- Host permissions: needed to inspect visible page image sources after user action.
