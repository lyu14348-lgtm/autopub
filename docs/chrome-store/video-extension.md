# Chrome Store Draft: Video Creative Analyzer

## Short Description
Analyze public or authorized videos for hook, structure, selling points, and CTA.

## Long Description
AutoPub Video Creative Analyzer helps marketers and content teams inspect public or authorized video links and generate structured creative notes for hook, pacing, selling points, emotional triggers, and CTA patterns.

## Permissions Rationale
- `activeTab`: inspect current page video metadata after user action.
- `scripting`: support current-page inspection.
- `storage`: save session and settings.
- Host permissions: detect visible public video metadata.

## Screenshot Checklist
- Manual URL input.
- Page video detection.
- Creative analysis result.
- Credits/upgrade state.
